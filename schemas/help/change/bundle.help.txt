Workspace package JSON contract

Command:
  grafana-util workspace package --dashboard-export-dir ./dashboards/raw --alert-export-dir ./alerts/raw --output-file ./workspace-package.json

kind:
  grafana-utils-sync-source-bundle

Top-level keys:
- kind
- schemaVersion
- toolVersion
- summary
- resources
- providers
- secretPlaceholders
- alertArtifacts
- resources[]
  - kind
  - identity
  - sourcePath

Notes:
- `workspace package` writes the contract to `--output-file`.
- It does not switch render modes with `--output-format`.
- Add `--also-stdout` only when you intentionally want both file output and stdout.
